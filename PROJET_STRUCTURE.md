# Structure du Projet InterShop Mobile

## Vue d'ensemble

Application mobile React Native (Expo) répliquant toutes les fonctionnalités d'alibaba-clone.

## Architecture

### Navigation (Bottom Tabs)
1. **Accueil** - Produits, restaurants, hôtels, profils de rencontre
2. **Catégories** - Navigation par catégories
3. **Messagerie** - Chat en temps réel
4. **Panier** - Gestion du panier
5. **Mon InterShop** - Profil, portefeuille, commandes

### Fonctionnalités Principales

#### 1. Authentification
- Inscription/Connexion (Email + Mot de passe)
- Vérification email (code à 6 chiffres)
- Vérification tél
